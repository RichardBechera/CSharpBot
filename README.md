# CSharpBot

This is discord bot written in C#. Right now it does nothing.

You need to create config.json file and fill in your data.
```
{
  "discord": {
    "token": "BOT TOKEN HERE",
    "appId": "CLIENT ID HERE",
    "appSecret": "CLIENT SECRET HERE",
    "CommandPrefix":  "/" 
  }
}
```
